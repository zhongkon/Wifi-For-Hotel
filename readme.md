## About 

Software นี้ทำขึ้นเพื่อใช้ควบคุมการใช้งาน wifi internet ภายในโรงแรมพัฒนาโดยใช้ Larvel 6 ใช้งานร่วมกับ Mikrotik + Freeradius3

## How to install

 git clone https://github.com/zhongkon/Wifi-For-Hotel.git</br>
composer install</br>
